# Anime Kanji - Aprenda Japonês com Animes 🎌

Um sistema completo de aprendizado de japonês inspirado em animes, utilizando SRS (Spaced Repetition System) para maximizar a retenção de Kanji, palavras e frases no contexto de animes.

## 🚀 Funcionalidades

- **Sistema SRS Inteligente**: Algoritmo de repetição espaçada para otimizar o aprendizado
- **Três tipos de conteúdo**: Kanji, Palavras e Frases contextualizadas em animes
- **Dashboard personalizado**: Acompanhe seu progresso, XP e streak diário
- **Sistema de níveis**: Progrida através dos níveis desbloqueando novo conteúdo
- **Autenticação completa**: Registro, login e proteção de rotas
- **Design responsivo**: Funciona perfeitamente em desktop e mobile
- **UI inspirada em animes**: Cores vibrantes e animações suaves

## 🛠️ Stack Tecnológica

- **Frontend**: Next.js 14 (App Router), TypeScript, TailwindCSS
- **Backend**: Supabase (Auth + PostgreSQL)
- **Deploy**: Vercel
- **Estado e formulários**: React Hook Form + Zod
- **UI Components**: Lucide React para ícones

## 📋 Pré-requisitos

- Node.js 18+ 
- Conta no Supabase (gratuita)
- Conta na Vercel (gratuita)
